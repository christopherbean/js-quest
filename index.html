Welcome to JS Quest, a turn based combat simulator built entirely with JavaScript and JQuery. The last fighter standing is declared the victor! If you'd like to play again please refresh your browser.

This game is still under development - new features and code revisions will be added over time as I continue to practice and learn more about frontend programming. After the initial launch plans include: different opponent types, more engaging combat mechanics, wave based combat, a leveling and saving system. Press start to begin!
